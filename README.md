# SnapSight 📸

A **Cyberpunk-themed AI Photo Analyzer** that gives you a detailed description of anything in a photo — objects, places, scenes, text, mood, and more.

Built with **Llama 4 Scout Vision** via **Groq API** (100% free, no credit card needed).

## Features

- 📷 Upload any photo or use your camera live
- 🔍 Identifies objects, locations, people, animals, text, signs
- 🎨 Detects colors, mood, and atmosphere
- 🏷️ Generates smart keyword tags
- 📋 Copy full analysis to clipboard
- 🔴 Cyberpunk 2077 inspired red-on-black UI

## Tech Stack

- Pure HTML, CSS, JavaScript (no frameworks)
- [Groq API](https://console.groq.com) — Llama 4 Scout Vision model
- Fonts: Barlow Condensed, Rajdhani, Share Tech Mono

## Setup

1. Clone the repo
   ```bash
   git clone https://github.com/Dhairya2306/snapsight.git
   cd snapsight
   ```

2. Open `index.html` in your browser

3. Get a **free Groq API key** from [console.groq.com/keys](https://console.groq.com/keys) (no credit card needed)

4. Paste your key in the app and start analyzing photos!

## Usage

- Click **Upload File** to pick a photo from your device
- Click **Use Camera** to take a photo with your webcam
- Click **Analyze with AI** to get a full structured report
